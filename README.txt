AutoSubtitle
AviUtl2 用 自動字幕生成プラグイン
Version: 0.1.0

============================================================
■ 対応環境
============================================================

・AviUtl2（最新版推奨）
・Windows 10 / 11（64bit）
・Python 3.12（推奨）
・faster-whisper
・ffmpeg（任意）

※ 本プラグインには Python / faster-whisper / ffmpeg は同梱されていません。


============================================================
■ 初回セットアップ（導入方法）
============================================================

1. AutoSubtitle.dll を AviUtl2 の Plugin フォルダに配置してください。

2. AviUtl2 を起動します。

3. メニューから AutoSubtitle を開きます。

4. 「実行環境」タブにて、必要に応じて Python 実行ファイルを指定します。
   ・未指定の場合、一般的なインストールパスから自動検出を試みます。

5. 設定は自動的に保存され、次回起動時に復元されます。


============================================================
■ 使い方
============================================================

1. AviUtl2 で字幕を作成したい音声・動画を読み込みます。

2. AutoSubtitle ウィンドウを開きます。

3. 各タブで必要に応じて設定を行います。
   ・認識プロファイル
   ・テキスト整形
   ・挿入設定 など

4. 「字幕生成」ボタンを押すと、
   ・音声認識
   ・字幕生成
   ・タイムラインへの自動挿入
   が順に行われます。


============================================================
■ ffmpeg について（任意）
============================================================

・ffmpeg を指定すると、音声の前処理（ノイズ除去・音量正規化など）が有効になります。
・ffmpeg を指定しない場合、音声前処理は行われません。

※ ffmpeg は必須ではありません。
※ ffmpeg を使用する場合は、各自で入手し、ライセンス条件を遵守してください。


============================================================
■ ログについて（エラー報告用）
============================================================

・「詳細ログを有効にする」を ON にした場合のみ、詳細なログが出力されます。
・エラー発生時は、以下のフォルダにログが出力されます。

C:\ProgramData\aviutl2\Plugin\AutoSubtitle\Logs\

（Python 関連のログは Logs\Python フォルダ内に出力されます）
